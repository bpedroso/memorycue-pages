Gerencie suas tarefas mensais com facilidade e nunca mais perca um compromisso importante.

Instalar no Chrome

Funcionalidades Principais

Gerenciamento de Tarefas

Adicione tarefas com título, observações e data específica do mês. Organize suas atividades de forma simples e eficiente.

Notificações

Receba lembretes para suas tarefas pendentes e nunca mais perca um compromisso importante.

Sincronização

Suas tarefas são sincronizadas automaticamente entre todos os seus dispositivos usando sua conta Google.

Como Usar

Clique no ícone da extensão na barra de ferramentas
Use o botão "+" para adicionar novas tarefas
Marque as tarefas como concluídas usando o checkbox
Use o botão de lápis para editar tarefas
Use o botão de lixeira para excluir tarefas
